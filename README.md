# Supervised Learning Assignment

## Project Title

Predicting Premium Subscription using Customer Behavior Data

## Dataset

E-commerce Shopper Behavior Dataset
Source: https://www.kaggle.com/datasets/dhrubangtalukdar/e-commerce-shopper-behavior-amazonshopify-based

## Task

Classification

## Target Variable

premium_subscription

## Description

This project builds machine learning models to predict whether a customer will subscribe to a premium service based on behavioral and marketing data.

## Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* Extra Trees
* SGD Classifier

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* ROC-AUC

## Files Included

* RQ1 to RQ7 notebooks
* Proposal document

## How to Run

1. Open notebooks in Kaggle or Jupyter
2. Upload dataset
3. Run all cells

## Outputs

* Tables saved as CSV
* Figures saved as PDF
